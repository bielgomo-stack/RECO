
RECO - Interactive Hotspots App (FINAL)

Instrucciones rápidas:
1. Descomprime este ZIP y sube el contenido al repositorio de GitHub.
2. Push a main -> Actions ejecutará el workflow y generará el APK (no firmado).
3. En Firebase Console, pega las reglas del archivo firestore.rules para seguridad básica.
4. Cambia el ADMIN_UID en lib/config.dart por tu UID de Firebase Auth para permitir edición.
